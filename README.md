# DIWALI_SALES_ANALYSIS_USING_PYTHON

# 🪔 Diwali Sales Analysis

## 📌 Overview

This project analyzes Diwali sales data using Python to understand customer behavior, sales trends, and product performance. The analysis focuses on customer demographics, purchasing patterns, and business insights.

---

## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset

The dataset contains customer transaction details such as:

| Column | Description |
|---------|-------------|
| User_ID | Unique customer ID |
| Cust_name | Customer name |
| Product_ID | Product ID |
| Gender | Customer gender |
| Age Group | Customer age category |
| Age | Customer age |
| Marital_Status | Marital status |
| State | Customer state |
| Zone | Region |
| Occupation | Customer occupation |
| Product_Category | Product category |
| Orders | Number of orders |
| Amount | Purchase amount |

---

## 🔍 Analysis Performed

- Data cleaning and preprocessing
- Gender-wise analysis
- Age group analysis
- State-wise sales analysis
- Marital status analysis
- Occupation analysis
- Product category analysis

---

## 📊 Key Insights

- Female customers contributed more to sales.
- Customers aged **26–35 years** made the highest purchases.
- **Uttar Pradesh, Maharashtra, and Karnataka** generated the most sales.
- Married women showed higher purchasing power.
- Customers from **IT, Healthcare, and Aviation** sectors spent the most.
- Food, Clothing, and Electronics were the most popular categories.
